This program requires a python 3.12 interpreter.
Additionally this program requires the python modules: flask, pandas, and openpyxl.

If you are running this on a windows machine, and do not have the required modules, please run the follwoing commands in your command line:
- pip install flask
- pip install pandas
- pip install openpyxl

Next, to run the program, please run flask_server.py and allow it to boot. 
From there you will be able to connect to it from a client via HTTP POST requests.
